# Análise do PIB a partir do transporte de veículos leves e pesados

Neste projeto, analiso como os volumes transportados de carros leves e pesados influenciam o PIB. 
Para isso, utilizo um modelo de Regressão Linear, treinando a máquina com as variáveis de entrada relacionadas ao transporte (BRP, SPP e SPT) e tendo o PIB como variável de saída.
